# Environment setup

## Create environment and run pytest tests
```bash
This project contains a suite of automated tests for validating data integrity in AdventureWorks2012 database. The tests are implemented using pytest and pymssql.
1. Clone this repository.
2. Install the required packages: pip install pytest pymssql
3. Create a user in MSSQL with read permissions.
4. Set up the database connection. If needed update the database connection parameters.
5. Execute tests, run the following command in your terminal: pytest -v tests.py

```
